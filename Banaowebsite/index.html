Who we are?

Banaowebsite.com is a developer friendly site which aid to develop his/her own website. It will cater the user different set of UI elements, ranging from jumbotron, navbar to radio button. The unique feature that separate it from its competitor website is text-to field view. This feature will allow the user to drag different UI elements to a blank area in order create a pseudo page that look exactly he wanted his webpage to look. It enables developer to access the shortest way to plan the aesthesis of the webpage.
